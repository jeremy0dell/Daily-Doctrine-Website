# Font & Asset Licensing Notes

## Fonts

### Georgia (Brand / Display Font)
- **Status**: No licensing concern
- **Reason**: Georgia is a core web font, pre-installed on Windows, macOS, iOS, and Android. It is freely available for web use without any license or font file hosting.
- **Usage**: App icon "DD" mark, wordmark SVGs, hero quote display
- **Web fallback stack**: `Georgia, "Times New Roman", "Noto Serif", serif`

### SF Pro (UI Font)
- **Status**: Apple-only, NOT licensed for web use
- **Reason**: SF Pro is Apple's proprietary system font. It renders automatically on Apple devices via `-apple-system` / `BlinkMacSystemFont` but cannot be hosted or embedded for cross-platform web use.
- **Web fallback stack**: `-apple-system, BlinkMacSystemFont, "SF Pro Display", "SF Pro Text", "Helvetica Neue", Helvetica, Arial, sans-serif`
- **Note**: The fallback stack ensures Apple users see SF Pro natively, while other platforms get Helvetica Neue or Arial. No font files are served.

## App Icon / Mark

- **Status**: Original work, owned by Daily Doctrine
- **Generated by**: `Scripts/generate-icon.swift` (uses CoreGraphics + CoreText)
- **Contains**: Georgia-Bold "DD" text on dark navy background with gradient effects
- **No third-party assets**: The icon is entirely programmatically generated from system fonts and drawing primitives

## OG Image

- **Status**: Original work, derived from app icon
- **Generated by**: `Tools/generate-og-image.swift`
- **Same licensing as app icon**: Georgia-Bold text on brand background colors

## Summary

No third-party font files, images, or licensed assets are included in this asset package. All assets are either:
1. Original programmatic artwork (icon, OG image)
2. System fonts referenced by name (Georgia, SF Pro via system stack)
3. Derived resize/reformats of original artwork (mark PNGs, favicons)
